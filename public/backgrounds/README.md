# 背景图片资源

此目录用于存储网站的背景图片资源。

## 图片命名规范

- `home-light.jpg` - 首页浅色模式背景
- `home-dark.jpg` - 首页深色模式背景
- `category-light.jpg` - 分类页浅色模式背景
- `category-dark.jpg` - 分类页深色模式背景
- `article-light.jpg` - 文章页浅色模式背景
- `article-dark.jpg` - 文章页深色模式背景
- `search-light.jpg` - 搜索页浅色模式背景
- `search-dark.jpg` - 搜索页深色模式背景
- `about-light.jpg` - 关于页浅色模式背景
- `about-dark.jpg` - 关于页深色模式背景

## 图片规格建议

- 格式：JPG（当前使用）、PNG、WebP
- 分辨率：1920x1080 或更高
- 文件大小：< 500KB
- 风格：与网站科技感设计风格协调
